# Rate Chicks 🐣

A fun (and slightly questionable) web app that allows users to rate profiles/images based on different criteria. Built as a simple full-stack project to explore user interaction, UI design, and basic rating systems.

---

## 🚀 Features

- 📸 Upload or display profiles/images
- ⭐ Rate based on predefined criteria
- 📊 Dynamic rating updates
- 🧠 Simple scoring logic
- 🎨 Clean and responsive UI

## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/pratyushere/rate-chicks.git
````

2. Navigate to the project folder:

```bash
cd rate-chicks
```

3. Install dependencies:

```bash
npm install
```

4. Run the project:

```bash
npm start
```

---

## 🧪 How It Works

* Users view profiles or images
* They assign ratings based on given parameters
* The system calculates and displays average scores
